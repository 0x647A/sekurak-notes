# Practical Python

This directory contains selected Python code written during the **"Practical Python – 10 projects step by step"** course conducted by **Gynvael Coldwind** on the [sekurak.pl](https://sekurak.pl/seria-10-szkolen-pythona-z-gynvaelem-praktycznie-i-bez-owijania-w-bawelne/) platform.

## About the course

The course offered by **Gyn** and **Sekurak** focuses on practical Python programming. Details and registration: https://python.sekurak.pl.

## Course projects

The course includes 10 practical projects:

1. **How much code did we write?** - Application that scans project directories (file system, text files)
2. **Helper agent** - Simple AI agent with OpenAI API (HTTP API and AI)
3. **Brute-force!** - Hash cracking with multithreading and multiprocessing
4. **Large files and bad connections** - Tools for file integrity verification
5. **Markdown converter** - Markdown to HTML parser and converter
6. **Cybersecurity tools** - Port scanner, brute-forcing, enumeration
7. **Recursive website cloning** - Creating website backups
8. **Home network monitoring** - Web panel for device monitoring
9. **Window interface** - GUI for console tools
10. **Simple game** - Game in PyGame

## Skills acquired during the course

- Starting your own programming projects
- File system operations
- Simple network operations and web backend
- Working with HTTP and APIs (including LLM)
- Text data processing
- Multithreading vs multiprocessing
- Creating window interfaces
- Game development basics

## Requirements

- **Python**: 3.11+
- **Dependencies** (vary by project):
  - `fastapi`, `uvicorn` — REST API project
  - `pydantic` — data validation (used with FastAPI)
  - `requests` — HTTP requests
  - `beautifulsoup4` — HTML parsing
